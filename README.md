# Eitaa-AI-Product-Crawler
Eitaa AI Product Crawler is an AI-powered distributed crawling pipeline designed to:  1.  Discover shop channels on Eitaa 2.  Validate them using AI 3.  Crawl and extract products 4.  Process data asynchronously using Redis Queue 5.  Provide semantic search using embeddings 6. Extracts structured product entities using LLM
